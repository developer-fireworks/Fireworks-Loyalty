# Customer Modules

The Customer Modules section provides comprehensive tools for managing your loyalty program members and their data.

## Overview

Through the customer management settings, administrators can perform various operations to manage customers effectively.

## Module Components

The Customer Modules consist of four main areas:

| Component | Description |
|-----------|-------------|
| **Audience List** | Create and manage segmented customer lists |
| **Customer Management** | Add, edit, and manage individual customers |
| **Migration** | Import and migrate customer data |
| **Rank Settings** | Configure membership tiers and ranks |

## Key Features

### Customer Data Management
- Create customers manually
- Update customer information
- Delete customer records
- Send password reset links
- Search and filter customers

### Filter Parameters

The system allows viewing users based on:

| Parameter | Description |
|-----------|-------------|
| Email | Customer email address |
| Name | Customer full name |
| Mobile Number | Contact phone number |
| Joining Date | Registration date |
| Tier/Rank | Membership level |
| Purchase of Rewards | Reward redemption history |
| Store Preference | Preferred shopping location |
| Old Member ID | Legacy system identifier |
| City | Customer location |
| And More | Additional custom fields |
