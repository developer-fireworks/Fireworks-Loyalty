# Edit/Delete Customer Information

Learn how to modify or remove customer records in the Loyalty CRM system.

## Overview

Administrators can edit customer information to keep records accurate or delete customers when necessary.

## Accessing Customer Records

Navigate to **Customer > Customer Management**

Use the search and filter features to locate the customer:
- Search by name, email, or phone number
- Apply filters for specific criteria

## Editing Customer Information

### Step 1: Locate the Customer

1. Use the search bar or filters to find the customer
2. Review the customer list to identify the correct record

### Step 2: Access Edit Mode

Click **Action > Edit** on the customer record.

### Step 3: Modify Information

Update any of the following fields as needed:

| Field | Description |
|-------|-------------|
| **Personal Information** | Name, DOB, Gender, NRIC |
| **Contact Details** | Email, Mobile, Address |
| **Membership Details** | Tier, Status, Card Number |
| **Additional Fields** | Custom fields as configured |

### Step 4: Save Changes

Click **Save** to apply the modifications.

## Deleting a Customer

### Step 1: Locate the Customer

Use search or filters to find the customer record.

### Step 2: Initiate Deletion

Click **Action > Delete** on the customer record.

### Step 3: Confirm Deletion

A confirmation prompt will appear. Confirm to proceed with deletion.

{% hint style="danger" %}
**Warning: Permanent Action**
Deleting a customer removes all associated data including:
- Transaction history
- Points balance
- Redemption history
- Communication records

This action cannot be undone.
{% endhint %}

## Password Reset

### Sending a Password Reset Link

1. Locate the customer record
2. Click **Action > Send Reset Password**
3. The customer will receive an email with reset instructions

## Bulk Operations

{% hint style="info" %}
**Bulk Updates:**
For updating multiple customers simultaneously, consider using:
- Import/Export functionality
- Migration tools
- Contact your administrator for bulk operations
{% endhint %}

## Audit Trail

All changes to customer records are logged, including:
- Who made the change
- When the change was made
- What was changed

## Best Practices

{% hint style="tip" %}
**Before Deleting:**
- Verify you have selected the correct customer
- Consider deactivating instead of deleting to preserve history
- Export customer data if needed for records
{% endhint %}

{% hint style="warning" %}
**Data Protection:**
Ensure compliance with data protection regulations when editing or deleting customer information.
{% endhint %}
