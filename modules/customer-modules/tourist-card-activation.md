# Tourist Card Activation

Learn how to activate membership cards for tourist customers.

## Overview

Tourist Card Activation allows staff to register and activate pre-issued membership cards for tourist customers.

## Accessing Tourist Card Activation

Navigate to **Customer > Customer Management > Tourist Card Activation**

## Activation Process

### Step 1: Navigate to Activation Page

Go to the Tourist Card Activation section in Customer Management.

### Step 2: Enter Card Information

| Field | Description |
|-------|-------------|
| **Card Number** | Enter the pre-printed card number |
| **Scan Option** | Use barcode scanner if available |

### Step 3: Enter Customer Details

Fill in the tourist's information:

| Field | Description | Required |
|-------|-------------|----------|
| **Full Name** | Tourist's complete name | Yes |
| **Email** | Valid email address | Yes |
| **Mobile Number** | Contact phone with country code | Yes |
| **Passport Number** | International identification | Yes |
| **Nationality** | Country of origin | Yes |
| **Date of Birth** | Birth date | No |
| **Gender** | Male/Female | No |

### Step 4: Set Activation Parameters

| Setting | Description |
|---------|-------------|
| **Activation Date** | When the card becomes active |
| **Validity Period** | Duration of card validity |
| **Initial Tier** | Starting membership level |

### Step 5: Complete Activation

Click **Activate** to complete the process.

## Card Features for Tourists

Tourist cards typically include:
- Welcome points bonus
- Special tourist-only promotions
- Limited validity period
- Multi-location acceptance

## Verification Steps

{% hint style="info" %}
**Before Activation:**
1. Verify the card number matches the physical card
2. Confirm tourist identification (passport)
3. Ensure contact details are accurate
{% endhint %}

## Common Issues

### Card Already Activated

{% hint style="warning" %}
If the card shows as already activated:
- Verify the card number
- Check if customer already exists in system
- Contact administrator if issue persists
{% endhint %}

### Invalid Card Number

{% hint style="warning" %}
If card number is not recognized:
- Check for typos in entry
- Ensure card is from valid batch
- Verify card format matches system requirements
{% endhint %}

## Post-Activation

After successful activation:
1. Customer receives confirmation (email/SMS)
2. Welcome points are credited
3. Card is ready for use at all participating outlets

## Best Practices

{% hint style="tip" %}
**Documentation:**
- Keep a record of passport details for verification
- Note the activation staff member
- Record any special requests or notes
{% endhint %}

{% hint style="info" %}
**Customer Communication:**
Explain to the tourist:
- Card validity period
- How to earn and redeem points
- Participating locations
- Customer service contact information
{% endhint %}
