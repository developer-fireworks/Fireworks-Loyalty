# Create Customer

Learn how to manually add new customers to the Loyalty CRM system.

## Overview

Administrators can manually create customer records when needed, such as for walk-in registrations or data entry from physical forms.

## Accessing Customer Creation

Navigate to **Customer > Customer Management**

## Creating a New Customer

<figure><img src="../../.gitbook/assets/unknown (1) (1) (1).png" alt=""><figcaption></figcaption></figure>

### Step 1: Access the Create Form

Click the **Create Customer** or **Manually Create Customer** button.

<figure><img src="../../.gitbook/assets/unknown (1) (1) (1) (1).png" alt=""><figcaption></figcaption></figure>

### Step 2: Enter Customer Information

Fill in the required customer details:

| Field             | Description              | Required |
| ----------------- | ------------------------ | -------- |
| **Full Name**     | Customer's complete name | Yes      |
| **Email**         | Valid email address      | Yes      |
| **Mobile Number** | Contact phone number     | Yes      |
| **NRIC/ID**       | Identity card number     | Varies   |
| **Date of Birth** | Customer's birth date    | No       |
| **Gender**        | Male/Female/Other        | No       |
| **Address**       | Customer's address       | No       |
| **City**          | City of residence        | No       |
| **Postcode**      | Postal code              | No       |
| **State**         | State/Province           | No       |
| **Country**       | Country of residence     | No       |

### Step 3: Set Membership Details

| Field               | Description                          |
| ------------------- | ------------------------------------ |
| **Membership Tier** | Assign initial rank/tier             |
| **Member ID**       | Auto-generated or custom ID          |
| **Join Date**       | Registration date                    |
| **Card Number**     | Physical card number (if applicable) |

### Step 4: Save the Customer

Click the **Save** or **Create** button to complete the registration.

## Tourist Card Activation

For tourist customers with pre-issued cards:

### Step 1: Navigate to Tourist Card Activation

Go to **Customer > Customer Management > Tourist Card Activation**

### Step 2: Enter Card Details

| Field                | Description                  |
| -------------------- | ---------------------------- |
| **Card Number**      | Tourist card number          |
| **Customer Details** | Fill in customer information |
| **Activation Date**  | Set activation date          |

### Step 3: Activate the Card

Click **Activate** to enable the tourist card.

## Data Validation

{% hint style="info" %}
**Duplicate Check:** The system automatically checks for duplicate entries based on:

* Email address
* Mobile number
* NRIC/ID number
{% endhint %}

{% hint style="warning" %}
**Required Fields:** Ensure all mandatory fields are completed before saving. The system will highlight missing required information.
{% endhint %}

## Best Practices

{% hint style="info" %}
**Data Quality:**

* Verify email addresses before saving
* Use consistent formatting for phone numbers
* Double-check NRIC entries for accuracy
{% endhint %}

{% hint style="info" %}
**Welcome Communications:** New customers can be automatically enrolled in welcome email campaigns. Check with your administrator for automated workflow settings.
{% endhint %}
