# Audience List

Learn how to create, manage, and utilize audience lists for targeted customer segmentation.

## Overview

Audience Lists allow you to segment your customers based on various criteria, enabling targeted communications and campaigns.

## Accessing Audience Lists

Navigate to **Customer > Customer Management > Audience List**

<figure><img src="../../.gitbook/assets/unknown (1).png" alt=""><figcaption></figcaption></figure>

## Audience List Interface

The audience list screen displays all created segments with the following options:

| Action              | Description                                   |
| ------------------- | --------------------------------------------- |
| **Edit**            | View and modify existing audience list        |
| **Delete**          | Remove the selected audience list permanently |
| **Create Audience** | Add a new audience list                       |
| **Export Audience** | Export the current audience list as CSV       |

## Creating an Audience List

### Step 1: Click Create Audience

Click the **Create Audience** button to open the creation form.

### Step 2: Configure Audience Details

| Field               | Description                                |
| ------------------- | ------------------------------------------ |
| **Audience Title**  | Enter a descriptive title for the list     |
| **Created On**      | Automatically populated with creation date |
| **Created By**      | Automatically populated with creator's ID  |
| **Filters**         | Apply filters to narrow down the audience  |
| **Columns**         | Choose which data fields to display        |
| **Report Listings** | Download the list in CSV format            |

### Step 3: Apply Filters

Use the filter options to segment your audience based on:

| Filter Category  | Options                             |
| ---------------- | ----------------------------------- |
| **Demographics** | Name, Email, Mobile, City           |
| **Membership**   | Tier/Rank, Joining Date, Status     |
| **Behavior**     | Purchase History, Points Balance    |
| **Preferences**  | Store Preference, Category Interest |

## Available Filters

The system provides extensive filtering capabilities:

* Date range filters (From/To dates)
* Status filters (Active/Inactive/Expired)
* Tier-based filters
* Store/location filters
* Transaction-based filters
* Points balance filters
* And many more custom options

## Managing Audience Lists

### Editing an Audience List

1. Click **Action > Edit** on the desired audience list
2. Modify the filters or column settings
3. Save your changes

### Deleting an Audience List

1. Click **Action > Delete** on the desired audience list
2. Confirm the deletion when prompted

{% hint style="warning" %}
**Warning:** Deleting an audience list is permanent and cannot be undone.
{% endhint %}

### Exporting an Audience List

1. Click **Export Audience** button
2. The list will be downloaded as a CSV file
3. Use the exported data for external analysis or campaigns

## Best Practices

{% hint style="info" %}
**Naming Convention:** Use descriptive names for audience lists that indicate the segment criteria (e.g., "Gold Tier - Active - KL Region")
{% endhint %}

{% hint style="info" %}
**Regular Updates:** Review and update audience lists periodically to ensure accuracy, especially for time-sensitive campaigns.
{% endhint %}
