# Facilities

Manage mall facilities and services available to members.

## Overview

The Facilities feature allows you to display and manage information about services and amenities available at your location.

## Accessing Facilities

Navigate to **Mall Management > Facilities**

## Facilities List

| Column | Description |
|--------|-------------|
| Facility Name | Display name |
| Category | Facility type |
| Location | Floor/area |
| Status | Active/Inactive |
| Actions | Edit/Delete |

## Adding a Facility

### Step 1: Click Add Facility

Navigate to **Facilities > Add/Create Facility**

### Step 2: Enter Details

| Field | Description | Required |
|-------|-------------|----------|
| **Facility Name** | Display title | Yes |
| **Category** | Facility type | Yes |
| **Description** | Detailed information | Yes |
| **Location** | Floor/area/unit | Yes |
| **Operating Hours** | Service hours | No |
| **Contact** | Phone/email | No |
| **Image** | Facility photo | No |
| **Status** | Active/Inactive | Yes |

### Step 3: Save Facility

Click **Save** to create the facility.

## Editing a Facility

1. Click **Action > Edit**
2. Modify information
3. Click **Save**

## Deleting a Facility

1. Click **Action > Delete**
2. Confirm deletion

## Facility Categories

| Category | Examples |
|----------|----------|
| **Services** | Customer service, concierge |
| **Amenities** | Prayer rooms, nursing rooms |
| **Financial** | ATMs, money changers |
| **Accessibility** | Wheelchair, stroller rental |
| **Parking** | Valet, motorcycle parking |
| **Others** | Lost & found, storage |

## Best Practices

{% hint style="tip" %}
**Accuracy:**
- Update hours for holidays
- Keep contact info current
- Verify locations after renovations
{% endhint %}

{% hint style="info" %}
**Helpful Details:**
Include practical information like:
- Nearest landmarks
- Access requirements
- Fees if applicable
{% endhint %}
