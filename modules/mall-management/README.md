# Mall Management

Manage mall facilities and amenities in the Loyalty CRM system.

## Overview

The Mall Management module allows you to configure and manage facilities available within your mall or venue.

## Module Components

| Component | Description |
|-----------|-------------|
| **Facilities** | Manage mall facilities and services |

## Quick Links

- [Facilities](facilities.md) - Add and manage facilities

## Accessing Mall Management

Navigate to **Mall Management** from the main menu.

## Facilities Overview

Facilities can include:
- Customer service centers
- Prayer rooms
- ATMs
- Parking services
- Wheelchair services
- Information counters
- Rest areas
- Baby care rooms

## Best Practices

{% hint style="tip" %}
- Keep facility information up-to-date
- Include accurate operating hours
- Add clear location directions
- Upload helpful images
{% endhint %}
