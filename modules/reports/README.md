# Reports

Comprehensive reporting and analytics for your loyalty program.

## Overview

The Reports module provides detailed analytics and reporting across all aspects of your loyalty program.

## Report Categories

| Category | Description |
|----------|-------------|
| **Campaign** | Campaign performance metrics |
| **Inventory** | Stock and redemption status |
| **Membership** | Member analytics |
| **Points** | Points transactions and balances |
| **Rewards** | Reward redemption data |
| **RSVP** | Event attendance reports |
| **Updates** | Content performance |

## Accessing Reports

Navigate to **Reports** from the main menu.

## Common Report Features

| Feature | Description |
|---------|-------------|
| **Date Filters** | Filter by date range |
| **Export** | Download as CSV/Excel |
| **Visualization** | Charts and graphs |
| **Scheduling** | Automated report delivery |

## Report Export

Most reports support:
- CSV export
- Excel export
- PDF export
- Scheduled email delivery
