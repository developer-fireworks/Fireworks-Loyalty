# RSVP Reports

Track event registration and attendance.

## Overview

RSVP Reports provide insights into event participation and performance.

## Available Reports

### RSVP Best Seller Report
Most popular events.

| Field | Description |
|-------|-------------|
| Event | Event name |
| Registrations | Sign-ups |
| Attendance | Actual attendees |
| Popularity | Ranking |

### RSVP Inventory Report
Event capacity utilization.

| Field | Description |
|-------|-------------|
| Event | Event name |
| Capacity | Total spots |
| Registered | Sign-ups |
| Available | Open spots |

### RSVP Report
Overall RSVP activity.

### RSVP Transaction History
Registration details.

| Field | Description |
|-------|-------------|
| Date | Registration date |
| Member | Registrant |
| Event | Event name |
| Status | Confirmed/Cancelled |

## Accessing Reports

Navigate to **Reports > RSVP**

## Best Practices

{% hint style="tip" %}
- Analyze attendance patterns
- Track no-show rates
- Optimize event timing
- Plan capacity based on history
{% endhint %}
