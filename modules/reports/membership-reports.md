# Membership Reports

Analyze member data, acquisition, and engagement.

## Overview

Membership Reports provide insights into your customer base and their activity.

## Available Reports

### Change of Card
Card replacement history.

### Change of Member Status
Status change tracking.

### Member Acquisition
New member sign-ups and sources.

| Metric | Description |
|--------|-------------|
| New Members | Registration count |
| Source | Acquisition channel |
| Conversion | Rate from lead |

### Membership Summary Report
Overall membership overview.

| Metric | Description |
|--------|-------------|
| Total Members | All registered |
| Active | Currently active |
| Inactive | Dormant accounts |
| By Tier | Distribution by rank |

### Purchase History
Transaction records by member.

### Rank/Tier Report
Tier distribution and movement.

| Metric | Description |
|--------|-------------|
| Tier | Membership level |
| Members | Count in tier |
| Upgrades | Tier promotions |
| Downgrades | Tier demotions |

### Referral Program
Referral activity and success.

### Membership Card Recruitment Details
Card enrollment tracking.

### Subscribers
Email/push subscription status.

### Usable Point Balance By Member
Individual point balances.

### User Journey
Member engagement patterns.

## Accessing Reports

Navigate to **Reports > Membership**

## Best Practices

{% hint style="tip" %}
**Member Analysis:**
- Track acquisition trends
- Monitor tier migration
- Identify at-risk members
- Measure engagement scores
{% endhint %}
