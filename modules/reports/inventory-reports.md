# Inventory Reports

Track reward inventory and redemption status.

## Overview

Inventory Reports help you monitor stock levels and redemption activity.

## Available Reports

### Gift Status Redemption @ POS
Point-of-sale redemption tracking.

| Field | Description |
|-------|-------------|
| Gift Item | Reward name |
| Redeemed | Quantity claimed |
| Pending | Awaiting pickup |
| Fulfilled | Completed |

### Inventory By Store
Stock levels by location.

| Field | Description |
|-------|-------------|
| Store | Location |
| Item | Reward |
| Available | Current stock |
| Low Stock | Alert status |

### Voucher Inventory Report
E-voucher and coupon stock.

| Field | Description |
|-------|-------------|
| Voucher | Voucher type |
| Total | Issued quantity |
| Used | Redeemed count |
| Available | Remaining |
| Expired | Unused expired |

## Accessing Reports

Navigate to **Reports > Inventory**

## Best Practices

{% hint style="tip" %}
- Monitor low stock alerts
- Plan restocking ahead of promotions
- Track expiring inventory
{% endhint %}
