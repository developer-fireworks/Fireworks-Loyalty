# Rewards Reports

Analyze reward redemption activity and performance.

## Overview

Rewards Reports provide insights into how rewards are being redeemed and their popularity.

## Available Reports

### Daily Check In Reward Report
Check-in reward distribution.

### E-Coupon Redemption Rules
Coupon usage tracking.

### Gift Redemption Status
Physical gift fulfillment tracking.

| Status | Description |
|--------|-------------|
| Pending | Awaiting fulfillment |
| Processing | Being prepared |
| Ready | Available for pickup |
| Completed | Successfully claimed |

### Reward Best Seller Report
Most popular rewards.

| Field | Description |
|-------|-------------|
| Reward | Item name |
| Redemptions | Total claims |
| Points | Points spent |
| Rank | Popularity ranking |

### Reward Inventory Report
Current reward stock levels.

### Reward Transaction Summary Report
Redemption activity overview.

| Metric | Description |
|--------|-------------|
| Period | Time frame |
| Redemptions | Total count |
| Points Spent | Total points |
| Popular Items | Top rewards |

### Reward Inventory History
Stock level changes over time.

### Rewards Report
Comprehensive reward data.

### Reward Transaction History
Detailed redemption records.

## Accessing Reports

Navigate to **Reports > Rewards**

## Best Practices

{% hint style="tip" %}
**Reward Analysis:**
- Identify best sellers
- Monitor inventory turnover
- Track fulfillment times
- Plan reward catalog updates
