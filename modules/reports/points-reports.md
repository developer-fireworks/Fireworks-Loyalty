# Points Reports

Track points transactions, balances, and history.

## Overview

Points Reports provide detailed analytics on point earning and redemption activity.

## Available Reports

### All Points Transactions
Complete transaction history.

| Field | Description |
|-------|-------------|
| Date | Transaction date |
| Member | Account holder |
| Type | Credit/Debit |
| Points | Amount |
| Description | Transaction note |

### Balance Report
Current point balances.

| Field | Description |
|-------|-------------|
| Member | Account holder |
| Available | Usable points |
| Pending | Processing |
| Expiring | Soon to expire |

### Historical Data Report
Long-term points trends.

### Point Credited/Debited Reports
Earning and spending breakdowns.

| Metric | Description |
|--------|-------------|
| Total Credited | Points earned |
| Total Debited | Points spent |
| Net Change | Balance movement |

### Points Transaction Summary
Aggregated transaction data.

| Metric | Description |
|--------|-------------|
| Period | Time frame |
| Earned | Total credited |
| Redeemed | Total debited |
| Expired | Points lost |

### Usable Point Balance VS Sales Report
Points liability vs. revenue.

## Accessing Reports

Navigate to **Reports > Points**

## Key Metrics

| Metric | Description |
|--------|-------------|
| Earning Rate | Points per transaction |
| Burn Rate | Points redeemed |
| Breakage | Unredeemed points |
| Liability | Outstanding points value |

## Best Practices

{% hint style="tip" %}
**Points Management:**
- Monitor liability levels
- Track earning vs burning ratio
- Manage expiration policies
- Forecast redemption patterns
{% endhint %}
