# Campaign Reports

Analyze campaign performance and effectiveness.

## Overview

Campaign Reports provide insights into how your marketing campaigns are performing.

## Available Reports

### Campaign Summary
Overall campaign performance metrics.

| Metric | Description |
|--------|-------------|
| Total Campaigns | Number of campaigns |
| Active Campaigns | Currently running |
| Members Reached | Unique participants |
| Points Awarded | Total points distributed |

### Promotion Rules Report
Performance of point multipliers and bonuses.

| Metric | Description |
|--------|-------------|
| Rule Name | Promotion identifier |
| Transactions | Qualifying transactions |
| Points Awarded | Total bonus points |
| Cost | Financial impact |

### Schedule Campaign Report
Scheduled campaign execution status.

| Metric | Description |
|--------|-------------|
| Scheduled | Pending campaigns |
| Completed | Finished campaigns |
| Success Rate | Execution success |

### QR Log Report
QR code scan activity.

| Metric | Description |
|--------|-------------|
| Total Scans | All scans |
| Unique Scanners | Distinct members |
| Top Locations | Popular scan points |

### QR Code Campaign Summary
QR campaign effectiveness.

| Metric | Description |
|--------|-------------|
| Campaign | QR campaign name |
| Scans | Total scans |
| Conversions | Completed actions |
| Points Distributed | Rewards given |

## Accessing Reports

Navigate to **Reports > Campaign**

## Filtering Options

- Date range
- Campaign type
- Status
- Merchant

## Export Options

- CSV
- Excel
- PDF
