# FAQ

Manage frequently asked questions for members.

## Overview

The FAQ section allows you to create and organize questions and answers to help members find information quickly.

## Accessing FAQ

Navigate to **News & Updates > FAQ**

## FAQ List

| Column | Description |
|--------|-------------|
| Question | FAQ title |
| Category | Topic grouping |
| Status | Active/Inactive |
| Order | Display sequence |
| Actions | Edit/Delete |

## Creating an FAQ

### Step 1: Click Create FAQ

Navigate to **FAQ > Create FAQ**

### Step 2: Enter Details

| Field | Description | Required |
|-------|-------------|----------|
| **Question** | The FAQ question | Yes |
| **Answer** | Detailed answer | Yes |
| **Category** | Topic category | No |
| **Sort Order** | Display position | No |
| **Status** | Active/Inactive | Yes |

### Step 3: Save FAQ

Click **Save** to publish the FAQ.

## Editing an FAQ

1. Click **Action > Edit**
2. Modify content
3. Click **Save**

## Deleting an FAQ

1. Click **Action > Delete**
2. Confirm deletion

## FAQ Categories

Organize FAQs by topic:
- Account & Registration
- Points & Earning
- Rewards & Redemption
- App & Technical
- General Information

## Best Practices

{% hint style="tip" %}
**Writing FAQs:**
- Use clear question format
- Provide complete answers
- Keep language simple
- Update based on common inquiries
{% endhint %}

{% hint style="info" %}
**Organization:**
- Group related questions
- Put most common questions first
- Review and update regularly
{% endhint %}
