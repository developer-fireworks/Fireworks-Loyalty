# Informational Pages

Create and manage static content pages.

## Overview

Informational Pages allow you to create custom content pages for your app or website.

## Accessing Informational Pages

Navigate to **News & Updates > Informational Pages**

## Creating an Informational Page

### Step 1: Click Create

Navigate to **Informational Pages > Create Informational Page**

### Step 2: Configure Page

| Field | Description |
|-------|-------------|
| **Title** | Page title |
| **URL Slug** | Page URL identifier |
| **Content** | Page body content |
| **Featured Image** | Header image |
| **Status** | Published/Draft |

### Step 3: Save Page

Click **Save** to publish.

## Editing/Deleting Pages

- **Edit:** Click Action > Edit to modify
- **Delete:** Click Action > Delete to remove

## Common Pages

- About Us
- How It Works
- Contact Us
- Store Directory
- Member Benefits
