# News and Promotions

Create and manage news articles, promotions, and member privileges.

## Overview

Publish news, promotional content, and member-exclusive offers to engage your audience.

## Accessing News and Promotions

Navigate to **News & Updates > News and Promotions**

## Content Types

| Type | Description |
|------|-------------|
| **News** | General announcements |
| **Promotions** | Special offers and deals |
| **Member Privilege** | Exclusive member benefits |

## Creating Content

### Step 1: Click Create

Navigate to **News and Promotions > Create News or Promotions or Member Privilege**

### Step 2: Configure Content

| Field | Description |
|-------|-------------|
| **Title** | Headline |
| **Type** | News/Promotion/Member Privilege |
| **Description** | Full content |
| **Featured Image** | Main image |
| **Start Date** | When to publish |
| **End Date** | When to expire |
| **Status** | Active/Inactive |

### Step 3: Save

Click **Save** to publish.

## Set as Banner

To feature content on the homepage:

1. Navigate to the content item
2. Click **Set as Banner**
3. Configure banner settings
4. Save

## Remove as Banner

1. Navigate to the banner item
2. Click **Remove as Banner**
3. Confirm removal

## Editing/Deleting Content

- **Edit:** Action > Edit
- **Delete:** Action > Delete

## Best Practices

{% hint style="tip" %}
- Use compelling headlines
- Include high-quality images
- Set appropriate date ranges
- Target content to relevant audiences
{% endhint %}
