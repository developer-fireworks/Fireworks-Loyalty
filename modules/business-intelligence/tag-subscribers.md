# Tag Subscribers

Apply and manage tags for member segmentation.

## Overview

Tag Subscribers allows you to categorize members using custom tags for targeted marketing and analysis.

## Accessing Tag Subscribers

Navigate to **Business Intelligent > Tag Subscriber**

## Applying Tags

### Step 1: Select Members

Use filters to select members:
- Search by name/email
- Use audience lists
- Apply custom filters

### Step 2: Choose Tag

Select existing tag or create new one.

### Step 3: Apply

Click **Apply Tag** to tag selected members.

## Managing Tags

### Tags for Subscribers

Navigate to **Business Intelligent > Tags for Subscribers**

| Column | Description |
|--------|-------------|
| Tag Name | Tag identifier |
| Members | Count of tagged |
| Created | Creation date |
| Actions | Edit/Delete |

### Creating Tags

1. Click **Create Tag**
2. Enter tag name
3. Add description
4. Save

### Editing/Deleting Tags

- **Edit:** Modify tag details
- **Delete:** Remove tag (does not delete members)

## Tag Strategies

### Behavioral Tags

```
High Spender - Members with purchases > RM1000/month
Frequent Visitor - 4+ visits per month
Reward Redeemer - Active in redemptions
```

### Lifecycle Tags

```
New Member - Joined within 30 days
At Risk - No activity in 60 days
Champion - High engagement score
```

### Campaign Tags

```
Summer 2025 Participant
Birthday Club Member
Referral Champion
```

## Best Practices

{% hint style="tip" %}
- Use clear, descriptive names
- Document tag criteria
- Review tag relevance regularly
- Use for targeted campaigns
{% endhint %}
