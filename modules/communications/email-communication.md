# Email Communication

Create and manage email templates and campaigns.

## Overview

Email Communication allows you to create professional email templates and send targeted campaigns.

## Accessing Email Communication

Navigate to **Communications > Email Communication**

## Creating Email Templates

### Step 1: Click Create

Navigate to **Email Communication > Create Email Templates**

### Step 2: Template Configuration

| Field | Description |
|-------|-------------|
| **Template Name** | Internal identifier |
| **Subject Line** | Email subject |
| **From Name** | Sender name |
| **Reply To** | Reply email address |
| **Content Type** | HTML/Plain text |

### Step 3: Design Content

| Element | Description |
|---------|-------------|
| **Header** | Logo and branding |
| **Body** | Main content |
| **Footer** | Unsubscribe and contact |
| **Personalization** | Dynamic fields |

### Step 4: Save Template

Click **Save** to store the template.

## Personalization Tags

| Tag | Output |
|-----|--------|
| `{first_name}` | Member's first name |
| `{points_balance}` | Current points |
| `{tier}` | Membership tier |
| `{expiring_points}` | Points expiring soon |

## Sending Emails

1. Select template
2. Choose audience
3. Preview content
4. Schedule or send immediately

## Best Practices

{% hint style="tip" %}
**Email Design:**
- Mobile-responsive layout
- Clear call-to-action
- Concise messaging
- Branded design
{% endhint %}

{% hint style="info" %}
**Deliverability:**
- Verify sender domain
- Monitor bounce rates
- Maintain list hygiene
{% endhint %}
