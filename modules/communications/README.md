# Communications

Manage member communications across multiple channels.

## Overview

The Communications module provides tools for creating and sending messages to members via email, SMS, push notifications, and in-app alerts.

## Communication Channels

| Channel | Description |
|---------|-------------|
| **Email** | Email campaigns and templates |
| **Push** | Mobile push notifications |
| **SMS** | Text message communications |
| **App Alerts** | In-app notifications |

## Communication Types

### Transactional
- Order confirmations
- Point notifications
- Redemption receipts

### Marketing
- Promotional offers
- News announcements
- Event invitations

### System
- Password resets
- Account alerts
- Expiration warnings
