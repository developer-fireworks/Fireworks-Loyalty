# Push Communication

Create and send mobile push notifications.

## Overview

Push Communication allows you to send instant notifications to member mobile devices.

## Accessing Push Communication

Navigate to **Communications > Push Communication**

## Creating Push Templates

### Step 1: Click Create

Navigate to **Push Communication > Create Push Templates**

### Step 2: Template Details

| Field | Description |
|-------|-------------|
| **Template Name** | Internal identifier |
| **Title** | Notification title |
| **Message** | Notification body |
| **Image** | Rich media (optional) |
| **Action** | Deep link destination |

### Step 3: Save Template

Click **Save** to store.

## Sending Push Notifications

1. Select template
2. Choose target audience
3. Set timing
4. Send or schedule

## Character Limits

| Element | Limit |
|---------|-------|
| Title | 50 characters |
| Message | 150 characters |

## Best Practices

{% hint style="tip" %}
- Keep messages brief
- Use compelling titles
- Time appropriately
- Don't over-send
{% endhint %}
