# SMS Communication

Create and send text message communications.

## Overview

SMS Communication allows you to send text messages to members.

## Accessing SMS Communication

Navigate to **Communications > SMS Communication**

## Creating SMS Templates

### Step 1: Click Create

Navigate to **SMS Communication > Create SMS Template**

### Step 2: Template Details

| Field | Description |
|-------|-------------|
| **Template Name** | Internal identifier |
| **Message** | SMS content |
| **Sender ID** | Displayed sender |

### Step 3: Save Template

Click **Save** to store.

## Character Limits

- Standard SMS: 160 characters
- Unicode SMS: 70 characters

## Sending SMS

1. Select template
2. Choose audience
3. Preview content
4. Send

## Best Practices
