# App Alerts

Create in-app notifications and alerts.

## Overview

App Alerts display within the mobile app or member portal.

## Accessing App Alerts

Navigate to **Communications > App Alerts**

## Creating Alerts

### Step 1: Click Create

Navigate to **App Alerts > Create New Alert**

### Step 2: Alert Configuration

| Field | Description |
|-------|-------------|
| **Alert Title** | Display title |
| **Message** | Alert content |
| **Type** | Info/Warning/Success |
| **Action** | Link destination |
| **Priority** | Display priority |

### Step 3: Set Visibility

| Setting | Description |
|---------|-------------|
| **Start Date** | When to show |
| **End Date** | When to hide |
| **Target** | All/Specific segments |

### Step 4: Save

Click **Save** to activate.

## Alert Types

| Type | Use Case |
|------|----------|
| **Info** | General announcements |
| **Warning** | Important notices |
| **Success** | Achievements, rewards |
| **Promo** | Promotional alerts |

## Best Practices

{% hint style="tip" %}
- Keep alerts relevant
- Don't overwhelm users
- Clear and actionable
- Time-limited display
{% endhint %}
