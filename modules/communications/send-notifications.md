# Send Notifications

Execute notification campaigns across channels.

## Overview

The Send Notifications feature allows you to distribute communications to your target audience.

## Accessing Send Notifications

Navigate to **Communications > Send Notifications**

## Sending Process

### Step 1: Select Channel

Choose communication channel:
- Email
- Push Notification
- SMS
- All channels

### Step 2: Select Template

Choose from saved templates.

### Step 3: Define Audience

| Option | Description |
|--------|-------------|
| **All Members** | Entire database |
| **Audience List** | Pre-defined segment |
| **Custom Filter** | Ad-hoc filtering |

### Step 4: Schedule

| Option | Description |
|--------|-------------|
| **Send Now** | Immediate delivery |
| **Schedule** | Future date/time |
| **Recurring** | Repeated schedule |

### Step 5: Review and Send

Preview, confirm, and execute.

## Tracking Delivery

| Metric | Description |
|--------|-------------|
| Sent | Total dispatched |
| Delivered | Successfully received |
| Opened | Email/push opened |
| Clicked | Links clicked |
| Bounced | Failed delivery |

## Best Practices

{% hint style="tip" %}
- Test before large sends
- Monitor delivery rates
- Track engagement
- Optimize timing
{% endhint %}
