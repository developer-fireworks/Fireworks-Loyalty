# Pending RSVP

Review and process pending event registrations.

## Overview

Pending RSVP shows all event registration requests awaiting approval.

## Accessing Pending RSVP

Navigate to **RSVP > Pending RSVP**

## Pending List

| Column | Description |
|--------|-------------|
| Event | Event name |
| Member | Registrant |
| Date | Registration date |
| Guests | Number of guests |
| Status | Pending |
| Actions | Approve/Reject |

## Processing RSVPs

### Approve Registration

1. Review request details
2. Click **Approve**
3. Confirmation sent to member

### Reject Registration

1. Review request
2. Click **Reject**
3. Select reason
4. Member notified

## Best Practices

{% hint style="tip" %}
- Process requests promptly
- Monitor capacity limits
- Send timely notifications
{% endhint %}
