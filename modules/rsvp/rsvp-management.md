# RSVP Management

Create and manage events for member registration.

## Overview

RSVP Management allows you to create events that members can register for.

## Accessing RSVP Management

Navigate to **RSVP > RSVP Management**

## Event List

| Column | Description |
|--------|-------------|
| Event Name | Title |
| Date | Event date |
| Capacity | Total slots |
| Registered | Current registrations |
| Status | Active/Closed |
| Actions | Edit/Delete/View |

## Creating an Event

### Step 1: Click Create

Navigate to **RSVP Management > Create**

### Step 2: Event Details

| Field | Description |
|-------|-------------|
| **Event Name** | Title |
| **Description** | Full details |
| **Date & Time** | When |
| **Location** | Where |
| **Capacity** | Max attendees |
| **Points Cost** | Points to register |
| **Guest Allowed** | Can bring guests |
| **Image** | Event image |

### Step 3: Settings

| Setting | Description |
|---------|-------------|
| **Registration Start** | When registration opens |
| **Registration End** | When registration closes |
| **Approval Required** | Auto/Manual approval |
| **Tier Restriction** | Eligible tiers |

### Step 4: Save

Click **Save** to create the event.

## Managing Events

- **Edit:** Modify event details
- **Delete:** Remove event
- **View Registrations:** See attendees
- **Export:** Download attendee list

## Best Practices

{% hint style="tip" %}
- Provide complete event information
- Set realistic capacity
- Allow adequate registration time
- Send reminders before event
{% endhint %}
