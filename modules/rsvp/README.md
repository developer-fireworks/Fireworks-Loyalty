# RSVP Management

Manage event reservations and RSVP functionality.

## Overview

The RSVP module allows members to register for events and administrators to manage event attendance.

## Module Components

| Component | Description |
|-----------|-------------|
| **Pending RSVP** | Review pending registrations |
| **RSVP Management** | Create and manage events |

## Quick Links

- [Pending RSVP](pending-rsvp.md)
- [RSVP Management](rsvp-management.md)

## Accessing RSVP

Navigate to **RSVP** from the main menu.

## Event Types

- Workshops
- Member events
- Exclusive launches
- Promotional events
- Community activities

## RSVP Workflow

```
Create Event → Members Register → Admin Approves → 
Event Occurs → Attendance Recorded
```

## Best Practices

{% hint style="tip" %}
- Set clear event details
- Manage capacity limits
- Send reminders
- Track attendance
{% endhint %}
