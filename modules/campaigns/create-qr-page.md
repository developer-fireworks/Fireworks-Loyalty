# Create QR Page

Learn how to create QR code landing pages for campaigns.

## Overview

QR Pages provide a destination for scanned QR codes, delivering content and rewards to members.

## Accessing Create QR Page

Navigate to **Campaign > QR Generator Pages > Create QR Page**

## Creating a QR Page

### Step 1: Basic Information

| Field | Description |
|-------|-------------|
| **Page Title** | Headline displayed to users |
| **Subtitle** | Secondary text |
| **Description** | Main content |
| **Status** | Active/Inactive |

### Step 2: Visual Content

| Field | Description |
|-------|-------------|
| **Header Image** | Banner/hero image |
| **Logo** | Brand logo |
| **Background** | Page background color/image |
| **Theme** | Color scheme |

### Step 3: Reward Configuration

| Setting | Description |
|---------|-------------|
| **Reward Type** | Points/Voucher/Both |
| **Points Amount** | Number of points |
| **Voucher** | Select voucher to award |
| **Instant Reward** | Award on scan |

### Step 4: Scan Rules

| Setting | Description |
|---------|-------------|
| **Scan Limit Per User** | Max scans per member |
| **Cooldown Period** | Time between scans |
| **Total Scan Limit** | Overall maximum |
| **Valid From** | Start date |
| **Valid Until** | End date |

### Step 5: Access Control

| Setting | Description |
|---------|-------------|
| **Member Only** | Require login |
| **Tier Restriction** | Specific tiers only |
| **New Members Only** | First-time scanners |

### Step 6: Call-to-Action

| Field | Description |
|-------|-------------|
| **Button Text** | CTA button label |
| **Button Action** | What happens on click |
| **Redirect URL** | Post-action destination |

### Step 7: Save and Generate

Click **Save** to create the page and generate QR code.

## Page Templates

### Welcome/Onboarding

```
Title: Welcome to Our Loyalty Program!
Reward: 100 welcome points
Limit: 1 per new member
CTA: Start Earning Now
```

### Event Check-In

```
Title: Thanks for Joining Us!
Reward: 50 event points
Limit: 1 per event
CTA: View Event Details
```

### Product Scan

```
Title: You Found a Bonus!
Reward: 25 product points
Limit: 1 per product
CTA: Explore More Products
```

### Location Visit

```
Title: Thanks for Visiting!
Reward: 30 visit points
Limit: 1 per day
CTA: See What's New
```

## Best Practices

{% hint style="tip" %}
**Page Design:**
- Keep messaging clear and concise
- Use engaging visuals
- Make rewards prominent
- Include clear CTA
{% endhint %}

{% hint style="info" %}
**Mobile Optimization:**
QR pages are typically viewed on mobile:
- Use mobile-friendly layouts
- Keep text readable
- Ensure fast loading
{% endhint %}

## After Creation

Once created:
1. Download the generated QR code
2. Test the scan process
3. Deploy in marketing materials
4. Monitor scan analytics
