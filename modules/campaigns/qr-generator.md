# QR Generator

Create QR codes for campaigns and engagement activities.

## Overview

The QR Generator allows you to create QR codes that members can scan to earn points, access promotions, or engage with your loyalty program.

## Accessing QR Generator

Navigate to **Campaign > QR Generator**

## QR Generator Features

| Feature | Description |
|---------|-------------|
| **QR Pages** | Landing pages for QR campaigns |
| **QR Codes** | Generate scannable codes |
| **Tracking** | Monitor scan activity |
| **Rewards** | Configure scan rewards |

## QR Generator Pages

### Step 1: Access QR Pages

Navigate to **Campaign > QR Generator Pages**

### Step 2: View QR Pages List

| Column | Description |
|--------|-------------|
| Page Name | Title of QR page |
| QR Code | Associated code |
| Status | Active/Inactive |
| Scans | Total scan count |
| Actions | Edit/Delete/View |

## Creating a QR Page

### Step 1: Click Create QR Page

Navigate to **QR Generator Pages > Create QR Page**

### Step 2: Configure Page Details

| Field | Description |
|-------|-------------|
| **Page Title** | Display title |
| **Description** | Page content |
| **Image/Banner** | Visual elements |
| **CTA Button** | Call-to-action |
| **Reward Type** | Points/Voucher |
| **Reward Amount** | Value to award |

### Step 3: Set Rules

| Setting | Description |
|---------|-------------|
| **Scan Limit** | Max scans per user |
| **Total Limit** | Max total scans |
| **Valid Period** | Start and end dates |
| **Member Only** | Require login |

### Step 4: Generate and Save

Click **Save** to create the QR page and generate code.

## QR Code Generation

### Step 1: Access QR Generator

Navigate to **Campaign > QR Generator**

### Step 2: Configure QR Code

| Field | Description |
|-------|-------------|
| **QR Name** | Code identifier |
| **Link To** | Destination (page/action) |
| **Size** | Code dimensions |
| **Format** | PNG/SVG/PDF |

### Step 3: Download/Export

Download the generated QR code for use in marketing materials.

## Use Cases

### Event Check-In

```
Location: Event venue
Action: Scan to check in
Reward: 50 event points
Limit: 1 scan per member
```

### Store Visit Bonus

```
Location: In-store display
Action: Scan to earn bonus
Reward: 20 bonus points
Limit: 1 per day
```

### Product Engagement

```
Location: Product packaging
Action: Scan for product info
Reward: 10 engagement points
Limit: Once per product
```

### Marketing Campaign

```
Location: Print/Digital ads
Action: Scan for promotion
Reward: Special voucher
Limit: Campaign total limit
```

## Best Practices

{% hint style="tip" %}
**Placement:**
- Ensure QR codes are easily accessible
- Provide clear scanning instructions
- Test codes before deployment
{% endhint %}

{% hint style="info" %}
**Design:**
- Use appropriate size for viewing distance
- Maintain quiet zone around code
- Include visual cue to scan
{% endhint %}

{% hint style="warning" %}
**Fraud Prevention:**
- Set reasonable scan limits
- Monitor for unusual patterns
- Use time-limited codes when appropriate
{% endhint %}
