# Daily Check-In Reward

Configure daily check-in rewards to encourage regular member engagement.

## Overview

Daily Check-In Rewards incentivize members to engage with the app or platform daily, building habit and increasing touchpoints.

## Accessing Daily Check-In

Navigate to **Campaigns > Daily Check In Reward**

## Configuration Options

### Basic Settings

| Setting | Description |
|---------|-------------|
| **Enable Check-In** | Turn feature on/off |
| **Reset Time** | When daily counter resets |
| **Streak Tracking** | Count consecutive days |

### Reward Structure

| Type | Description |
|------|-------------|
| **Fixed Reward** | Same points daily |
| **Progressive** | Increasing rewards |
| **Streak Bonus** | Bonus for consecutive days |
| **Weekly Completion** | Bonus for full week |

## Setting Up Daily Check-In

### Step 1: Enable Feature

Toggle the Daily Check-In feature to Active.

### Step 2: Configure Rewards

| Day | Points | Notes |
|-----|--------|-------|
| Day 1 | 5 | Standard daily |
| Day 2 | 5 | Standard daily |
| Day 3 | 10 | Mid-week bonus |
| Day 4 | 5 | Standard daily |
| Day 5 | 5 | Standard daily |
| Day 6 | 10 | Weekend bonus |
| Day 7 | 20 | Week completion bonus |

### Step 3: Set Streak Rules

| Setting | Description |
|---------|-------------|
| **Streak Reset** | When streak resets (miss a day) |
| **Streak Bonus** | Additional rewards for streaks |
| **Maximum Streak** | Cap for streak counting |

### Step 4: Save Configuration

Click **Save** to activate the check-in rewards.

## Reward Structures

### Linear Structure
```
Every day: 5 points
Weekly completion bonus: 50 points
```

### Progressive Structure
```
Day 1: 5 points
Day 2: 10 points
Day 3: 15 points
...
Day 7: 50 points + weekly bonus
```

### Streak-Based Structure
```
Days 1-7: 5 points/day
Days 8-14 (2nd week streak): 10 points/day
Days 15-30: 15 points/day
30+ day streak: 20 points/day
```

## Member Experience

Members can:
1. Open app/website daily
2. Click "Check In" button
3. Receive instant reward
4. Track their streak progress
5. View upcoming rewards

## Best Practices

{% hint style="tip" %}
**Engagement Design:**
- Make check-in easy (one-click)
- Show streak progress visually
- Celebrate milestones
- Send reminders for active streakers
{% endhint %}

{% hint style="info" %}
**Balance Rewards:**
- Not too high (maintains value)
- Not too low (worth the effort)
- Progressive rewards encourage consistency
{% endhint %}

{% hint style="warning" %}
**Streak Fairness:**
Consider allowing:
- Grace period (miss one day)
- Makeup check-ins (limited)
- Clear communication of rules
{% endhint %}

## Reports

- User retention impact
