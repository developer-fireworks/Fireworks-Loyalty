# POS Tags

Create and manage tags for POS transactions.

## Overview

POS Tags allow you to label and categorize transactions for reporting and analysis.

## Accessing POS Tags

Navigate to **POS Purchase > POS Tags**

## Tags List

| Column | Description |
|--------|-------------|
| Title | Tag name |
| Code | Tag identifier |
| Remark | Notes |
| Actions | Edit/Delete |

## Creating POS Tags

### Step 1: Click Add

Navigate to **POS Tags > New POS Tag**

### Step 2: Enter Details

| Field | Description |
|-------|-------------|
| **Title** | Tag display name |
| **Code** | Unique identifier |
| **Remark** | Description/notes |

### Step 3: Save

Click **Save** to create the tag.

## Editing/Deleting Tags

- **Edit:** Action > Edit to modify
- **Delete:** Action > Delete to remove

## Tag Use Cases

| Tag | Purpose |
|-----|---------|
| **Promotion** | Promotional transactions |
| **Return** | Returned items |
| **Staff** | Staff purchases |
| **Void** | Voided transactions |

## Best Practices

{% hint style="tip" %}
- Use clear, descriptive titles
- Maintain consistent coding
- Document tag purposes
- Review usage regularly
{% endhint %}
