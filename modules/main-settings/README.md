# Settings Overview

Configure core loyalty program settings including points management and rate configurations.

## Overview

The Main Settings module provides administrative controls for fundamental loyalty program configurations.

<figure><img src="../../.gitbook/assets/unknown (16).png" alt=""><figcaption></figcaption></figure>

## Module Components

| Component               | Description                          |
| ----------------------- | ------------------------------------ |
| **Add/Minus Points**    | Manual point adjustments             |
| **Point Configuration** | Earning and redemption rules         |
| **Table Rate**          | Spending-to-points conversion tables |

## Quick Links

* [Add/Minus Points](add-minus-points.md) - Manually adjust member points
* [Point Configuration](point-configuration.md) - Configure earning rules
* [Table Rate](table-rate.md) - Set spending conversion rates

## Accessing Main Settings

Navigate to **Main Settings** from the main menu.

## Key Settings Overview

### Point Earning Rules

* Base earning rate (points per spend)
* Bonus earning conditions
* Category multipliers
* Time-based multipliers

### Point Redemption Rules

* Redemption rate (points to value)
* Minimum redemption threshold
* Maximum redemption limits
* Expiration policies

### Table Rates

* Spending tier configurations
* Progressive earning rates
* Category-specific rates

## Best Practices

{% hint style="info" %}
**Configuration Changes:**

* Document all changes made
* Test changes in staging environment first
* Communicate changes to stakeholders
* Monitor impact after implementation
{% endhint %}

{% hint style="warning" %}
**Impact Assessment:** Changes to point settings affect:

* Member earning rates
* Program liability
* Financial projections
* Member expectations
{% endhint %}
