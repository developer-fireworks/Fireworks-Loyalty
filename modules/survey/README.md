# Survey Module

Create and manage member surveys for feedback collection.

## Overview

The Survey Module allows you to create surveys to gather member feedback and insights.

## Module Components

| Component | Description |
|-----------|-------------|
| **Survey Edit/View** | Manage existing surveys |
| **Create Survey** | Build new surveys |
| **View Survey** | Preview surveys |
| **Survey Statistics** | Analyze responses |
| **Show Responses** | View individual answers |
| **All Surveys** | Survey listing |

## Accessing Survey Module

Navigate to **Survey** from the main menu.

## Survey Types

| Type | Use Case |
|------|----------|
| **Satisfaction** | Customer experience |
| **Feedback** | Service/product feedback |
| **Research** | Market research |
| **Event** | Post-event feedback |

## Best Practices

{% hint style="tip" %}
**Survey Design:**
- Keep surveys concise
- Use clear questions
- Mix question types
- Offer incentives for completion
{% endhint %}
