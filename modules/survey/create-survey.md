# Create Survey

Build surveys to collect member feedback.

## Overview

Create customized surveys with various question types.

## Accessing Create Survey

Navigate to **Survey > Create Survey**

## Creating a Survey

### Step 1: Basic Details

| Field | Description |
|-------|-------------|
| **Title** | Survey name |
| **Description** | Introduction text |
| **Status** | Active/Inactive |

### Step 2: Add Questions

| Question Type | Description |
|---------------|-------------|
| **Multiple Choice** | Select one option |
| **Checkbox** | Select multiple |
| **Text** | Short answer |
| **Textarea** | Long answer |
| **Rating** | Scale rating |
| **Dropdown** | Select from list |

### Step 3: Configure Settings

| Setting | Description |
|---------|-------------|
| **Required** | Mandatory answer |
| **Anonymous** | Hide respondent |
| **Points Reward** | Incentive points |

### Step 4: Save

Click **Save** to publish survey.

## Question Best Practices

{% hint style="tip" %}
- Be clear and specific
- Avoid leading questions
- Limit required questions
- Test before publishing
{% endhint %}
