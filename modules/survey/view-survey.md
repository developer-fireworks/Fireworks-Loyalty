# View Survey

Preview and review survey content.

## Overview

View Survey allows you to preview how surveys appear to respondents.

## Accessing View Survey

1. Navigate to **Survey > Survey Edit/View**
2. Click **View** on the desired survey

## Preview Features

- Full survey rendering
- Question display
- Response flow
- Mobile preview

## Survey Status

| Status | Description |
|--------|-------------|
| **Draft** | Not published |
| **Active** | Collecting responses |
| **Closed** | No longer accepting |
| **Archived** | Stored for records |
