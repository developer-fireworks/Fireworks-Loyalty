# Survey Statistics

Analyze survey responses and results.

## Overview

Survey Statistics provides analytics on survey responses.

## Accessing Statistics

Navigate to **Survey > Survey Statistics**

## Available Metrics

| Metric | Description |
|--------|-------------|
| **Total Responses** | Number of completions |
| **Completion Rate** | Started vs finished |
| **Average Time** | Time to complete |
| **Response Trend** | Over time |

## Question Analysis

For each question:
- Response distribution
- Charts and graphs
- Percentage breakdown
- Average ratings (for rating questions)

## Show Responses

View individual responses:
1. Navigate to **Show Responses**
2. Filter by date or respondent
3. View detailed answers

## Exporting Data

Export options:
- CSV download
- Excel export
- PDF report

## Best Practices

{% hint style="tip" %}
- Review responses regularly
- Look for patterns
- Act on feedback
- Share insights with team
{% endhint %}
