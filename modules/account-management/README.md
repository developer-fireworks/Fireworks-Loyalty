# Account Management

The Account Management module provides essential functionality for user authentication and profile management within the Loyalty CRM system.

## Overview

This module covers all aspects of managing your account access and personal information in the system.

## Features

| Feature | Description |
|---------|-------------|
| Login | Secure authentication to access the CRM system |
| Change Password | Update your login credentials |
| Update Profile | Modify your personal information |
| Logout | Securely exit the system |
| Forgot Password | Recovery options for lost credentials |
| Sign Up | New user registration process |

## Security Notes

{% hint style="info" %}
Always ensure you log out of the system when accessing from shared computers.
{% endhint %}

{% hint style="warning" %}
Your login credentials should never be shared with others. Contact your administrator if you suspect unauthorized access to your account.
{% endhint %}
