# Login

Learn how to access the Loyalty CRM system through the login process.

## Prerequisites

Before logging in, ensure you have:

* A valid email address or phone number registered in the system
* Your password credentials
* Access to the CRM URL provided by the Fireworks Team

## Step-by-Step Guide

### Step 1: Access the Login Page

<figure><img src="../../.gitbook/assets/unknown (1) (1) (1) (1) (1).png" alt=""><figcaption></figcaption></figure>

Navigate to the URL provided by the Fireworks team. You will see the login screen.

### Step 2: Enter Your Credentials

The login form contains the following fields:

| Field                  | Description                                                     |
| ---------------------- | --------------------------------------------------------------- |
| **Email / Phone / IC** | Enter your registered email address, phone number, or IC number |
| **Password**           | Enter the password associated with your account                 |
| **reCAPTCHA**          | Click the checkbox to verify you are human                      |

### Step 3: Complete Authentication

1. Fill in your credentials in the appropriate fields
2. Complete the reCAPTCHA verification
3. Click the **Login** button

### Step 4: Dashboard Access

Upon successful login, you will be redirected to the dashboard screen.

<figure><img src="../../.gitbook/assets/unknown (2).png" alt=""><figcaption></figcaption></figure>

## Dashboard Overview

Once logged in, the dashboard provides quick access to:

* **Today** - Filter by today's report data
* **Yesterday** - Filter by yesterday's report data
* **Last 7 days** - Filter by last 7 days report data
* **Last 30 days** - Filter by last 30 days report data
* **From & To** - Custom date range filter
* **Submit** - Apply the selected date filter

## Troubleshooting

### Common Issues

{% hint style="warning" %}
**Invalid Credentials Error**

* Verify your email/phone/IC is correct
* Check that CAPS LOCK is not enabled
* Ensure you're using the correct password
{% endhint %}

{% hint style="info" %}
**Forgot Your Password?** Click the "Forgot Password" link on the login page to initiate the password recovery process. See [Forgot Password](forgot-password.md) for detailed instructions.
{% endhint %}
