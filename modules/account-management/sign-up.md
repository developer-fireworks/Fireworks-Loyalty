# Sign Up

Learn how to register as a new user in the Loyalty CRM system.

## Overview

New users can register for access to the Loyalty system through the sign-up process.

## Prerequisites

Before signing up, prepare the following information:

* Valid business name or username
* NRIC (National Registration Identity Card) number
* Valid email address
* Mobile phone number
* Industry information
* Domain/Site URL (if applicable)

## Step-by-Step Guide

### Step 1: Access Sign Up Page

1. Go to the login page
2. Click the **"Sign Up"** button

<figure><img src="../../.gitbook/assets/unknown (7).png" alt=""><figcaption></figcaption></figure>

### Step 2: Fill in Registration Form

Complete all required fields in the registration form:

| Field                | Description                                              |
| -------------------- | -------------------------------------------------------- |
| **Business Name**    | Enter your username or business name                     |
| **NRIC**             | Enter your NRIC number for identity verification         |
| **Email**            | Enter a valid email address                              |
| **Phone**            | Enter your mobile phone number                           |
| **Industry**         | Specify the industry you are involved in                 |
| **Site URL**         | Provide a custom site URL linked to your domain          |
| **Password**         | Create a secure password for your account                |
| **Confirm Password** | Re-enter the same password to confirm                    |
| **Terms Checkbox**   | Confirm you have read and agreed to terms and conditions |

### Step 3: Complete Registration

1. Review all entered information for accuracy
2. Tick the checkbox to agree to terms and conditions
3. Click the **Register** button to complete registration

### Form Actions

| Button       | Action                                       |
| ------------ | -------------------------------------------- |
| **Reset**    | Clears all entered information to start over |
| **Register** | Submits the registration form                |

## Password Requirements

{% hint style="info" %}
**Create a Strong Password:**

* Minimum 8 characters recommended
* Mix of uppercase and lowercase letters
* Include numbers and special characters
* Avoid using personal information
{% endhint %}

## After Registration

Once registered:

1. You may receive a confirmation email
2. Your account may require administrator approval
3. Login with your new credentials once approved

{% hint style="warning" %}
**Approval Process:** New registrations may require approval from an administrator before access is granted. Contact the Fireworks Team if you need expedited access.
{% endhint %}

## Troubleshooting

### Common Issues

**Email Already Registered**

* If you've previously registered, use the [Forgot Password](forgot-password.md) feature
* Contact administrator if you believe this is an error

**Invalid NRIC Format**

* Ensure NRIC is entered in the correct format
* Contact support if issues persist
